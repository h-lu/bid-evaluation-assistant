# 辅助评标专家系统 —— 详细调研报告

> 调研目标：为给定投标书进行合规分析、模拟专家打分，为专家提供便捷与建议  
> 调研时间：2026年2月20日

---

## 执行摘要

### 核心定位

**本系统定位为评标专家的"智能助手"，而非替代专家**。系统通过AI技术为专家提供：
1. **合规分析自动化**：自动完成资格审查、符合性审查等重复性工作
2. **评分建议智能化**：对客观分自动计算，对主观分提供参考建议
3. **评审过程便捷化**：快速定位、对比分析、风险提醒

### 关键发现

1. **政策明确支持**：195号文明确支持"智能辅助评标"，定位为"辅助专家开展评审"
2. **技术成熟可行**：文件智能解析、自动评分、偏离分析等技术已有成功案例
3. **人机协作是核心**：AI初评+专家复核的模式已成为行业共识
4. **专家痛点明确**：文件多、时间紧、主观判断难、容易遗漏

### 1.1 传统评标流程

```

│                     评标专家传统工作流程                         │
├──────────┬──────────────────────────────────────────────────────┤
│ 阶段     │ 工作内容                                              │
├──────────┼──────────────────────────────────────────────────────┤
│ 准备阶段 │ • 领取招标文件、投标文件                               │
│          │ • 熟悉评标办法和评分标准                               │

├──────────┼──────────────────────────────────────────────────────┤
│ 初步审查 │ • 资格审查（资格条件核对）                             │
│          │ • 符合性审查（实质性响应检查）                         │
│          │ • 判定无效投标                                        │
├──────────┼──────────────────────────────────────────────────────┤
│ 详细评审 │ • 技术评审（技术方案、参数响应）                       │

├──────────┼──────────────────────────────────────────────────────┤
```

### 1.2 评标专家核心痛点

#### 痛点1：文件量大、时间紧

| 场景 | 具体表现 |
|-----|---------|
| **标书页数多** | 单个投标文件100-1000页不等 |
| **文件数量多** | 一个项目3-10家投标单位 |
| **评审时间短** | 需在1-3天内完成评审 |
| **关键信息分散** | 资质、业绩、参数等信息分散在不同章节 |

**专家原话**：
> "一份标书上百页，专家逐行审，效率低易疏漏"

#### 痛点2：资格审查工作量大

| 审查项 | 工作量 | 易错点 |
|-------|-------|-------|
| 营业执照 | 核对有效期、经营范围 | 分支机构资质 |
| 资质证书 | 核对等级、有效期 | 过期或降级 |
| 业绩证明 | 核对合同金额、时间 | 业绩造假 |
| 财务报告 | 核对审计报告、数据 | 数据矛盾 |
| 信用记录 | 查询信用中国、执行信息 | 遗漏不良记录 |
| 人员证书 | 核对职称、资格证书 | 证书过期 |

#### 痛点3：技术参数对比困难

- **参数数量多**：医疗设备可能有50-200项技术参数
- **响应方式复杂**：正偏离、负偏离、无偏离需逐一判定
- **证明材料多**：检测报告、彩页、说明书需核对
- **★号条款关键**：实质性参数负偏离直接导致废标

#### 痛点4：主观评分难把握

| 问题 | 影响 |
|-----|-----|
| 评分标准模糊 | "技术方案先进性"难以量化 |
| 专家主观差异 | 不同专家对同一方案评分差异大 |
| 缺乏参照 | 难以横向对比不同投标单位 |
| 记忆负担重 | 难以记住所有投标单位的方案细节 |

#### 痛点5：评分偏离风险

- **横向偏离**：同一专家对不同投标单位评分尺度不一
- **纵向偏离**：不同专家对同一投标单位评分差异大
- **极端评分**：畸高畸低评分可能引发质疑投诉

**政策要求**（195号文）：
> "提示评标专家进行复核确认并修改完善"评分不一致、分值计算错误、专家打分偏离度过大等问题

### 2.2 资格审查详细清单

#### 2.2.1 法定资格条件（政府采购法）

| 序号 | 审查项 | 审查内容 | 证明材料 |
|-----|-------|---------|---------|
| 1 | 独立承担民事责任 | 法人资格、分支机构授权 | 营业执照 |
| 2 | 商业信誉 | 无重大违法记录 | 信用中国截图 |
| 3 | 财务状况 | 健全的财务会计制度 | 财务审计报告 |
| 4 | 履行合同能力 | 设备、技术能力 | 相关证明材料 |
| 5 | 税收社保 | 依法缴纳税收和社会保障资金 | 完税证明、社保缴纳证明 |
| 6 | 其他要求 | 特定行业资质 | 资质证书 |

#### 2.2.2 特定资格条件（项目要求）

| 类型 | 审查内容 | 示例 |
|-----|---------|-----|
| **资质要求** | 行业准入资质 | 医疗器械经营许可证、生产许可证 |
| **业绩要求** | 类似项目经验 | 近3年同类设备销售合同 |
| **人员要求** | 专业技术人员 | 项目经理资质、技术人员职称 |
| **财务要求** | 资金实力 | 注册资本、银行资信证明 |
| **信用要求** | 信用等级 | AAA级信用企业证书 |

### 2.3 符合性审查详细清单

#### 2.3.1 形式审查

| 序号 | 审查项 | 审查标准 | 废标情形 |
|-----|-------|---------|---------|
| 1 | 投标文件签署 | 法定代表人签字或盖章 | 未签字盖章 |
| 2 | 投标文件密封 | 符合招标文件要求 | 密封破损 |
| 3 | 投标文件格式 | 符合招标文件模板 | 格式严重不符 |
| 4 | 投标保证金 | 金额、形式符合要求 | 未缴纳或不足 |
| 5 | 投标有效期 | 满足招标文件要求 | 有效期不足 |
| 6 | 投标报价 | 唯一报价、无选择性 | 多个报价方案 |

#### 2.3.2 实质性响应审查

**实质性要求识别**：
- 招标文件中标注"★"号的条款
- "必须"、"应当"、"实质性要求"等关键词
- 技术参数中的最低性能指标
- 商务条款中的核心要求

**医疗器械常见实质性要求**：

| 类别 | 实质性要求示例 |
|-----|---------------|
| **技术参数** | 设备分辨率、精度、量程等核心指标 |
| **注册证** | 医疗器械注册证在有效期内且覆盖招标范围 |
| **质保期** | 质保期不少于招标文件要求 |
| **交货期** | 能在规定时间内交货安装 |
| **付款方式** | 接受招标文件付款条件 |
| **售后服务** | 在当地有服务机构或承诺设立 |

#### 2.3.3 技术偏离判定

**偏离类型定义**：

| 偏离类型 | 定义 | 示例 |
|---------|-----|-----|
| **正偏离** | 优于招标要求 | 质保期3年（要求2年） |
| **无偏离** | 完全符合要求 | 参数完全匹配 |
| **负偏离** | 不满足招标要求 | 分辨率低于要求 |

**判定规则**：
- ★号条款负偏离 → **废标**
- 一般技术参数负偏离 → **扣分**
- 需提供证明材料支持正偏离

## 第三部分：专家打分体系

### 3.1 评分类型划分

```
┌──────────────────────────────────────────────────────────────┐
│                      综合评分法结构                           │
├──────────────────────────────────────────────────────────────┤
│                                                              │
│   总分100分 = 价格分(30-60分) + 技术分(20-40分) +            │
│              商务分(10-20分) + 其他分(0-10分)                │
│                                                              │
├──────────────────────────────────────────────────────────────┤
│   技术分构成：                                                │
│   • 客观分（可量化）：技术参数响应、资质等级、业绩数量        │
│   • 主观分（需判断）：技术方案先进性、实施能力、创新性        │
│                                                              │
└──────────────────────────────────────────────────────────────┘
```

### 3.2 客观分评分标准（AI可自动计算）

#### 3.2.1 技术参数响应评分

**评分方式1：逐项扣分法**

| 参数类型 | 分值 | 评分规则 |
|---------|-----|---------|
| ★号关键参数 | 40分 | 每项5分，负偏离扣5分 |
| 重要参数(▲) | 30分 | 每项3分，负偏离扣3分 |
| 一般参数 | 30分 | 每项1分，负偏离扣1分 |

**评分方式2：符合度计算法**

```
技术参数得分 = 基础分 × (响应参数数量 / 总参数数量)

示例：
- 总参数100项
- 响应95项（无偏离85项，正偏离10项）
- 基础分30分
- 得分 = 30 × (95/100) = 28.5分
```

#### 3.2.2 资质业绩评分

| 评分项 | 评分标准 | 分值 |
|-------|---------|-----|
| 资质等级 | 甲级10分，乙级7分，丙级5分 | 10分 |
| 业绩数量 | 每提供1个同类业绩得2分，最高10分 | 10分 |
| 财务状况 | 营业收入≥1亿得5分，<1亿得3分 | 5分 |
| 认证体系 | ISO9001/ISO13485等，每项2分 | 5分 |

#### 3.2.3 价格分计算

**基准价法**（常用）：

```
投标报价得分 = (评标基准价 / 投标报价) × 价格分权重

其中：
- 评标基准价 = 有效投标报价的算术平均值
- 或：最低有效投标报价作为基准价
```

**低价优先法**：

```
投标报价得分 = [1 - (投标报价 - 最低价) / 最低价] × 价格分权重
```

### 3.3 主观分评分标准（AI提供建议）

#### 3.3.1 技术方案评审（通常20-30分）

| 评分维度 | 分值 | 评分标准 | AI分析要点 |
|---------|-----|---------|-----------|
| **方案完整性** | 5分 | 方案全面，无重大遗漏 | 检查方案章节完整性 |
| **技术先进性** | 5分 | 采用先进技术，有创新点 | 提取技术亮点关键词 |
| **实施可行性** | 5分 | 实施计划合理，风险可控 | 分析工期安排逻辑性 |
| **针对性** | 5分 | 针对本项目需求定制 | 检测方案模板化程度 |
| **可操作性** | 5分 | 操作步骤清晰，易于实施 | 评估流程描述详细度 |

**AI辅助评分建议**：
- **方案亮点提取**：自动识别技术方案中的创新点
- **横向对比**：与其他投标单位方案进行对比
- **风险提示**：识别方案中的潜在风险点
- **参考评分**：基于历史数据给出参考分值范围

#### 3.3.2 售后服务评审（通常5-10分）

| 评分维度 | 分值 | 评分标准 | AI分析要点 |
|---------|-----|---------|-----------|
| **服务网点** | 3分 | 本地有服务网点或承诺设立 | 核实网点真实性 |
| **响应时间** | 3分 | 故障响应时间符合要求 | 对比招标要求 |
| **质保内容** | 2分 | 质保范围全面，增值服务多 | 检查质保条款 |
| **培训方案** | 2分 | 培训计划详细，针对性强 | 评估培训内容 |

### 3.4 评分偏离预警机制

#### 3.4.1 偏离类型定义

| 偏离类型 | 判定标准 | 预警级别 |
|---------|---------|---------|
| **横向偏离** | 同一专家对不同投标单位评分差异>20% | 黄色预警 |
| **纵向偏离** | 不同专家对同一投标单位评分差异>30% | 橙色预警 |
| **极端评分** | 评分畸高（>95分）或畸低（<40分） | 红色预警 |

#### 3.4.2 系统预警功能

**实时提醒**：
- 专家提交分数时，系统自动计算偏离度
- 偏离度超过阈值时弹出提醒
- 提示专家复核确认

**提示内容示例**：
> "您对投标单位A的技术方案评分为95分，显著高于其他专家的平均分75分（偏离度26.7%），请确认评分合理性。"

### 4.3 人机协作流程设计

#### 4.3.1 协作原则

**AI负责**：
- ✓ 重复性、标准化工作（信息提取、资格审查）
- ✓ 计算性工作（客观分计算、偏离度计算）
- ✓ 提示性工作（风险预警、偏离提醒）

**专家负责**：
- ✓ 判断性工作（主观分评定、废标决策）
- ✓ 最终决策（评分确认、评审结论）
- ✓ 责任承担（签署评审报告）

#### 4.3.2 评审流程

```
阶段1：AI初评（自动完成）
├── 文件解析（5分钟）
├── 合规审查（自动）
├── 客观分计算（自动）
└── 生成初评报告

阶段2：专家详评（人工完成）
├── 复核合规审查结果
├── 评定主观分（参考AI建议）
├── 添加评审意见
└── 提交评分

阶段3：偏离检查（系统辅助）
├── 计算评分偏离度
├── 预警异常评分
├── 提示专家复核
└── 修正评分

阶段4：报告生成（AI辅助）
├── 汇总评分结果
├── 生成评标报告
├── 专家签字确认
└── 完成评审
```
